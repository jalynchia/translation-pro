## 核心功能

- **双向翻译**：中文译英文，英文译中文，自动检测源语言。
- **语域适配**：根据学术、营销、法律、社媒等不同场景调整语调，确保译文得体。
- **本地化转换**：自动适配日期、数字与计量单位的本地格式。
- **消除翻译腔**：翻译初稿完成后，对照 humanizer 规范重新润色，消除机器翻译感与 AI 常用词汇。

## 目录结构

- `SKILL.md`：核心流程与翻译约束规则。
- `references/`：用于后处理润色的去 AI 痕迹指南。
  - `humanizer-zh.md`：中文去 AI 痕迹规范。
  - `humanizer.md`：英文去 AI 痕迹规范。

## 交互流程

1. **选择输出方式**：首轮交互会询问是在对话中直接展示译文，还是存为 `.md` 文件（如保存至 `~/Downloads/`）。
2. **术语与语境确认**：若输入文本存在明显歧义（例如多义词且缺乏上下文），会先向用户确认再开始翻译。
3. **分批处理**：每次只处理当前输入的文本，不主动询问是否继续，减少对话干扰。
